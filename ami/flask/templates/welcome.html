Artificial Modular Intellegence

Here is where AMI is descibed on how to use. We will cover a lot of information with links to more information.

AMI is an AI that consits of multiple parts

  • A Flask server. In part the visual UI and in part the app that controls the AI.
  • A tkinter GUI. This runs on the main thread of the main process and controls what the user sees.
  • A python AI with multiple Headspaces for seperation of thought and responsibility.
  • Headsapces, which are basically LLM agents with filesystem access. These are communitiy driven add-ons for this platform. Instructions to build your own Headspace can be found here(no link yet)
